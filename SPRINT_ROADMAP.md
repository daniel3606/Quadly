## 스프린트 6: 확장 기능 (2주)

### 목표
추가 기능 중 1개 선택하여 MVP 구현

### 옵션
1. **채팅**: 1대1 채팅, 그룹 채팅
2. **중고 거래**: 상품 등록, 검색, 거래
3. **식당 메뉴**: 오늘의 메뉴 조회

### 작업 항목 (선택된 기능에 따라)
- [ ] 선택된 기능의 데이터 모델 설계
- [ ] API 구현
- [ ] UI 구현
- [ ] 신고 및 차단 기능 적용
- [ ] 운영 정책 수립

### 완료 기준
- 신규 기능도 신고 차단이 완비됨
- 성능과 스팸 리스크가 통제됨
- 사용자 테스트 통과

## 성공 지표

- 7일 잔존율: 40% 이상
- 일간 활성 사용자 수: 증가 추세
- 게시글 대비 댓글 비율: 0.3 이상
- 신고 처리 시간 평균: 24시간 이내
- 코스 리뷰 수: 학기당 코스당 평균 2개 이상
- 시간표 생성 수: 가입 사용자의 60% 이상
